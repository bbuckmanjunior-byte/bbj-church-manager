Fresh BBJ Digital Church Manager

This is a fresh scaffolded minimal Java Servlet app. Files:
- `src/com/bbj/db/DatabaseConnection.java`
- `src/com/bbj/servlets/*.java` (Login, Logout, Announcement, Event, Member, Sermon)
- `WebContent/*.html`, `WebContent/css/style.css`, `WebContent/WEB-INF/web.xml`
- `database/church_schema.sql`

Follow steps in parent README to compile and deploy to Tomcat.
